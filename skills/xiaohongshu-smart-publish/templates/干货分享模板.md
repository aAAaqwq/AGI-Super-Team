# 小红书排版模板：干货分享型

## 标题公式
`身份+数字+利益` 或 `人群+痛点+解决方案`

示例:
- "打工人必看🔥5个AI工具让你准时下班"
- "程序员3年经验💻这些效率工具我全用过"
- "省钱达人亲测💰5个免费工具吊打付费版"

## 正文模板

```
{emoji开头} {核心观点/钩子}

{emoji1} {第一点：痛点/问题}
{详细说明，2-3句话，短句为主}

{emoji2} {第二点：解决方案/工具}
{具体操作步骤或使用方法}
- 步骤1
- 步骤2
- 步骤3

{emoji3} {第三点：进阶技巧}
{额外价值信息，让读者觉得"学到了"}

{emoji4} {第四点：注意事项}
{避坑指南或限制说明}

{emoji5} {第五点：总结/福利}
{收尾引导：收藏/关注/评论}

---
{emoji收藏} 觉得省时省力别忘了点赞收藏！
{emoji关注} 关注我，每天分享实用干货~
{emoji评论} 有问题评论区见，有问必答！

#标签
{精准标签1} {精准标签2} {精准标签3} {泛标签4} {泛标签5} {泛标签6}
```
