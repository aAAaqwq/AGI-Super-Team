# 小红书发布流程与排版规范

**平台**: https://creator.xiaohongshu.com/publish/publish  
**调研日期**: 2026-03-17

---

## 1. 发布入口

- **网页端**: https://creator.xiaohongshu.com/publish/publish
- **类型选择**: 上传图片/视频/图文轮播
- 支持类型: 图文笔记、视频笔记、图文轮播

---

## 2. 排版规则表

### 2.1 标题规范

| 规则项 | 限制 | 备注 |
|--------|------|------|
| **字数上限** | 20字 | 短笔记和长笔记均限20字 |
| **最小字数** | 建议8字以上 | 过短影响搜索收录 |
| **emoji** | ✅ 推荐 | 每个标题1-3个emoji |
| **关键词密度** | 标题+正文+结尾 | 增加搜索收录 |
| **标点** | 少用句号 | 用空格或emoji分隔 |

**标题爆款公式**:
```
人群定位 + 关键词 + 数字 + 悬念词 + emoji
例: "打工人必看🔥5个效率工具让你准时下班"
```

### 2.2 正文规范

| 规则项 | 限制 | 备注 |
|--------|------|------|
| **短笔记上限** | 1000字 | 普通笔记 |
| **长笔记(内测)** | 1000字以上 | 需内测资格 |
| **短笔记最小字数** | 100字(新规) | 低于100字影响推荐 |
| **长笔记最小字数** | 600字 | 内测长文要求 |
| **emoji** | ✅ 每段1-2个 | 不过度使用 |
| **换行** | 双换行分段 | 单换行不生效 |
| **空行** | 每段间空一行 | 提升可读性 |

### 2.3 标签/话题规范

| 规则项 | 限制 | 备注 |
|--------|------|------|
| **标签格式** | #话题名 | 井号+无空格 |
| **数量** | 建议3-8个 | 不超过10个 |
| **位置** | 文末或穿插正文 | 结尾集中排列 |
| **话题类型** | 精准话题+泛话题 | 至少2个精准话题 |

### 2.4 封面图规范

| 规则项 | 限制 | 备注 |
|--------|------|------|
| **尺寸** | 3:4竖版(1080×1440) | 最优比例 |
| **格式** | JPG/PNG | JPG优先 |
| **大小** | < 20MB | 单张限制 |
| **主标题** | 3-5个字 | 字体清晰醒目 |
| **配色** | ≤3种 | 品牌感 |
| **人物/产品占比** | 60%+ | 突出主体 |
| **引导元素** | 箭头/圆圈/高亮 | 增加点击率 |

### 2.5 图片规范

| 规则项 | 限制 | 备注 |
|--------|------|------|
| **数量** | 1-18张 | 图文笔记 |
| **单张大小** | < 20MB | - |
| **格式** | JPG/PNG/WebP | - |
| **推荐尺寸** | 1080×1440(3:4) | 竖版最优 |
| **宽屏** | 1920×1080(16:9) | 横屏图 |

## 3. 发布步骤

### 3.1 网页端发布流程

```
1. 打开 https://creator.xiaohongshu.com/publish/publish
2. 选择笔记类型:
   - [上传图片] → 图文笔记
   - [上传视频] → 视频笔记
3. 上传媒体文件:
   - 点击上传区域 / 拖拽文件
   - 支持 JPG, PNG, WebP, MP4
4. 填写标题:
   - 最多20字
   - 建议含emoji+关键词+数字
5. 编写正文:
   - 建议600-1000字
   - 段落间空行
   - 适当emoji
6. 添加标签:
   - #话题名 格式
   - 建议3-8个
7. 选择发布设置:
   - [发布] → 立即发布
   - [存草稿] → 保存草稿
   - [定时发布] → 选择时间
```

### 3.2 草稿保存

- 草稿箱位置: 创作服务平台 → 草稿箱
- 草稿自动保存: 编辑过程中自动保存
- 草稿有效期: 无限制
- 草稿管理: 可编辑/删除/发布

## 4. CSS选择器/按钮定位参考

> 注: 以下基于creator.xiaohongshu.com页面结构，可能随版本变化

### 4.1 关键元素

| 元素 | CSS选择器/XPath | 说明 |
|------|-----------------|------|
| 上传区域 | `input[type="file"]` | 文件上传input |
| 标题输入 | `[placeholder*="标题"]` 或 `#title` | 标题输入框 |
| 正文输入 | `[contenteditable="true"]` 或 `.ql-editor` | 富文本编辑器(Quill) |
| 标签输入 | `.tag-input` 或 `[placeholder*="标签"]` | 话题输入框 |
| 发布按钮 | `button.publish-btn` 或 `[class*="publish"]` | 发布按钮 |
| 草稿按钮 | `button.draft-btn` 或 `[class*="draft"]` | 存草稿按钮 |
| 笔记类型选择 | `.note-type-selector` | 图片/视频选择 |
| 图片列表 | `.upload-list .item` | 已上传图片列表 |
| 删除图片 | `.upload-list .delete-btn` | 删除按钮 |

### 4.2 页面交互

```javascript
// 填写标题
document.querySelector('[placeholder*="标题"]').textContent = "标题内容";

// 填写正文(Quill编辑器)
document.querySelector('.ql-editor').innerHTML = "正文内容<br><br>第二段";

// 模拟输入标签
const tagInput = document.querySelector('.tag-input input');
tagInput.value = "话题名";
tagInput.dispatchEvent(new KeyboardEvent('keydown', {key: 'Enter'}));
```

## 5. 注意事项

### 5.1 新规要点(2025)

- **原创至上**: 短笔记≥100字，长笔记≥600字
- **AI内容标记**: 如使用AI生成，需标记
- **搬运限制**: 严禁搬运，需原创
- **水印规范**: 禁止其他平台水印

### 5.2 发布最佳时间

| 时间段 | 工作日 | 周末 |
|--------|--------|------|
| 早高峰 | 7:00-9:00 | 10:00-12:00 |
| 午间 | 12:00-14:00 | 15:00-17:00 |
| 晚高峰 | 18:00-22:00 | 20:00-23:00 |

### 5.3 排版技巧

1. **标题**: 数字+痛点 / 反差+悬念 / 身份+利益
2. **开头**: 前3行决定用户是否继续阅读
3. **正文**: 分段+空行+emoji，避免大段文字
4. **结尾**: 引导互动(提问/投票/关注)
5. **标签**: 精准话题在前，泛话题在后
6. **封面**: 大字+对比色+人物/产品

## 6. 自动化要点

- 编辑器类型: **Quill** 富文本编辑器
- 登录要求: **必须登录** (扫码/手机号)
- 反爬: 有滑块验证、频率限制
- 图片上传: 需先上传获取URL，再插入编辑器
- 草稿保存: 接口 POST 保存草稿数据
