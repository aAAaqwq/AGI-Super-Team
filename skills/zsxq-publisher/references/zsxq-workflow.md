# 知识星球发布流程与排版规范

**平台**: https://wx.zsxq.com (网页版) / APP端  
**调研日期**: 2026-03-17

---

## 1. 发布入口

- **网页端**: https://wx.zsxq.com → 选择星球 → 发帖
- **APP端**: 知识星球APP → 选择星球 → 写主题/提问
- **登录方式**: 
  - APP: 微信授权登录(仅微信)
  - 网页: 微信扫码 / 手机号登录

## 2. 排版规则表

### 2.1 主题/帖子规范

| 规则项 | 限制 | 备注 |
|--------|------|------|
| **正文字数** | 无硬性上限 | 支持长文，建议500-5000字 |
| **短消息** | 建议50-200字 | 快速互动 |
| **长文** | 建议500+字 | 深度内容 |
| **排版** | 富文本 / Markdown | 原生富文本，Markdown需插件 |
| **emoji** | ✅ 支持 | 原生emoji支持 |

### 2.2 提问规范

| 规则项 | 限制 | 备注 |
|--------|------|------|
| **提问类型** | 普通提问 / 匿名提问 / 付费提问 | 由星主设置 |
| **付费提问** | 星主定价 | 可设置价格 |
| **提问字数** | 建议100-500字 | 清晰描述问题 |

### 2.3 标签规范

| 规则项 | 限制 | 备注 |
|--------|------|------|
| **标签设置** | 星主设置菜单标签 | 相当于分类 |
| **标签数量** | 无硬性限制 | 建议5-15个 |
| **发帖时选标签** | 可选 | 选择对应标签分类 |
| **标签修改** | 可修改 | 但已关联的帖子不会批量更新 |

### 2.4 附件规范

| 规则项 | 限制 | 备注 |
|--------|------|------|
| **文件格式** | 支持常见办公格式 | 具体见下方 |
| **文件数量** | 单次上传可能有数量限制 | 一个主题可多次上传 |
| **文件大小** | 单文件≤50MB | 建议单个≤20MB |
| **视频格式** | MP4, MOV | - |
| **视频大小** | ≤1GB | 单个视频 |
| **视频时长** | ≤60分钟 | - |
| **图片格式** | JPG, PNG, GIF | - |
| **图片大小** | ≤20MB | 单张 |
| **文档格式** | PDF, DOC, DOCX, PPT, XLS, XLSX | - |
| **压缩包** | ZIP, RAR | 可能有审核 |

### 2.5 图片规范

| 规则项 | 限制 | 备注 |
|--------|------|------|
| **单次上传** | 1-9张 | 一次可选多张 |
| **图片大小** | ≤20MB/张 | - |
| **格式** | JPG, PNG, GIF | - |
| **宽屏/竖版** | 均支持 | 自适应显示 |

## 3. 发布步骤

### 3.1 网页端发帖流程

```
1. 打开 https://wx.zsxq.com
2. 登录(微信扫码/手机号)
3. 选择目标星球
4. 点击 [写主题] / 输入框
5. 编辑内容:
   a. 选择标签(可选): 从星主设置的标签中选择
   b. 编写正文:
      - 富文本编辑(加粗/引用/列表等)
      - 或粘贴Markdown内容
   c. 上传图片:
      - 点击图片图标
      - 支持多选上传
   d. 上传文件/视频:
      - 点击附件图标
      - 选择文件上传
   e. 选择匿名(如果星主允许)
6. [发布] → 立即发布
   [定时发送] → 选择时间(需星主功能)
   [存草稿] → 保存草稿
```

### 3.2 发帖类型

| 类型 | 说明 | 适用场景 |
|------|------|----------|
| **主题(长文)** | 深度文章 | 知识分享、教程、分析 |
| **短动态** | 简短消息 | 碎片化思考、日常分享 |
| **提问** | 向星主/成员提问 | 问题咨询、求助 |
| **匿名提问** | 匿名提问 | 敏感话题 |
| **打卡** | 日常打卡 | 习惯养成(星主开启) |
| **作业** | 布置作业 | 教学场景(星主开启) |

### 3.3 Markdown支持

知识星球原生编辑器为富文本，但支持以下方式使用Markdown:

1. **粘贴方式**: 从Markdown编辑器复制渲染后的HTML，粘贴到编辑器
2. **浏览器插件**: 使用第三方Chrome插件(如"星球伴侣"等)
3. **直接输入**: 部分Markdown语法可被识别:
   - `# 标题` → 可能被渲染为标题
   - `**粗体**` → 粗体
   - `> 引用` → 引用
   - `- 列表` → 列表

## 5. 注意事项

### 5.1 星主 vs 普通成员

| 功能 | 星主 | 普通成员 |
|------|------|----------|
| 发帖 | ✅ | ✅(需星主开放权限) |
| 设置标签/菜单 | ✅ | ❌ |
| 定时发布 | ✅ | ❌ |
| 设置付费提问 | ✅ | ❌ |
| 发布打卡/作业 | ✅ | ❌ |
| 禁止下载 | ✅ | ❌ |
| 屏蔽成员 | ✅ | ❌ |
| 内容保护设置 | ✅ | ❌ |

### 5.2 发表权限

- 星主可设置发表权限:
  - **全部成员可发表**: 所有人可发帖
  - **仅星主/管理员发表**: 普通成员只能评论
  - **审核后发表**: 发帖需星主审核

### 5.3 排版技巧

1. **长文结构**: 标题 → 摘要 → 分段 → 总结
2. **短动态**: 一段话+配图+话题标签
3. **提问**: 背景+问题+期望回答方向
4. **图片**: 重要信息截图+关键内容文字
5. **文件**: 提供PDF版本和源文件

### 5.4 限制与注意

| 注意项 | 说明 |
|--------|------|
| 付费星球 | 最低50元，续费折扣最低5折 |
| 退款 | 付费72小时内可退款 |
| 文件下载 | 星主可设置"禁止退款用户下载" |
| 内容保护 | 星主可限制新成员可查看的内容范围 |
| 违规 | 股债期基金币圈/宗教/迷信/ChatGPT类星球暂不支持 |

## 6. 自动化要点

- **框架**: 传统Web应用(非SPA)
- **登录**: 微信扫码 / 手机号
- **反爬**: 有频率限制，Cookie认证
- **发帖**: POST请求，需CSRF Token
- **文件上传**: 分片上传，需签名
- **图片上传**: 直接上传，返回URL
- **API**: 无公开API，需模拟操作
- **定时发布**: 仅星主可用
