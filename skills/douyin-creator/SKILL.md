# 抖音创作发布

## 排版规则

### 视频规格

| 项 | 规则 |
|----|------|
| 时长 | 15秒-60分钟（>15分需粉丝≥1000） |
| 格式 | MP4(H.264推荐)、MOV |
| 分辨率 | 竖屏1080×1920(9:16)最优，横屏1920×1080(16:9) |
| 帧率 | ≥30fps |
| 码率 | 2500-5000kbps |
| 大小 | ≤4GB |

### 图文规格

| 项 | 规则 |
|----|------|
| 图片数 | 2-35张（至少2张） |
| 格式 | JPG/PNG/WebP |
| 尺寸 | 竖版1080×1440(3:4)最优 |
| 大小 | ≤20MB/张 |

### 描述/标签

| 项 | 规则 |
|----|------|
| 描述 | ≤200字(含标签)，emoji适度(每2-3句1个) |
| 话题 | #话题名(无空格) 3-5个，精准在前泛话题在后 |
| @提及 | @相关账号增加互动 |
| 位置 | 可添加POI定位(本地内容推荐) |
| AIGC标注 | AI生成内容**必须标注** "内容由AI生成" |

## 内容适配模板

## 发布流程

### 视频发布

```bash
python scripts/douyin_publish.py \
  --type video \
  --file /path/to/video.mp4 \
  --description "视频描述文字" \
  --tags "话题1,话题2,话题3" \
  --cover /path/to/cover.jpg \
  --mode draft \
  --cookie-file ~/.douyin_cookies.json
```

### 图文发布

```bash
python scripts/douyin_publish.py \
  --type photo \
  --images "/path/to/img1.jpg,/path/to/img2.jpg" \
  --description "图文描述" \
  --tags "话题1,话题2" \
  --mode draft \
  --cookie-file ~/.douyin_cookies.json
```

### 草稿模式（默认安全）
所有发布默认 `--mode draft`。草稿可在创作者平台管理。

### 正式发布
`--mode publish` 直接发布。可附加 `--schedule "2026-03-20 20:00"` 定时发布（最远10天）。

## 算法要点

- 发布后**2小时内**进入赛马机制
- 权重排序：完播率 > 点赞率 > 评论率 > 转发率
- **前3秒**决定用户是否继续观看
- 话题标签影响推荐池分配
- 最佳发布时间：20:00-22:00（晚间黄金档）

## 错误处理

| 错误 | 处理 |
|------|------|
| 登录过期 | 提示重新扫码，headless=false |
| 滑块验证 | 暂停等待手动完成 |
| 视频转码失败 | 检查格式/分辨率，提示使用H.264编码 |
| 上传超时 | 大文件分片重试，指数退避(10s/30s/90s) |
| 图片≤1张 | 报错：图文至少需要2张图片 |
| 描述超长 | 截断到200字并警告 |
| 话题不存在 | 使用模糊匹配选择最近话题 |

## 发布前检查清单

- [ ] 视频：MP4 H.264，≤4GB，9:16竖屏
- [ ] 图文：≥2张图片，3:4竖版
- [ ] 描述 ≤200字
- [ ] 话题 3-5个
- [ ] AIGC内容已标注
- [ ] 无搬运/水印
- [ ] 封面已选择（视频从帧选取或自定义上传）
