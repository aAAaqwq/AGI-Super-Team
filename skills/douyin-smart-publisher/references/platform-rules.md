# 抖音平台规则详解

## 一、内容规范

### 1.1 禁止内容
- 违法违规内容（政治敏感、色情暴力等）
- 虚假信息、谣言
- 侵权内容（未经授权的音乐、影视片段）
- 诱导行为（刷量、诱导点赞/关注）
- 垃圾内容（重复搬运、重复内容）

### 1.2 限流原因
- 搬运内容（未注明来源）
- 诱导互粉互赞
- 频繁发布相似内容
- 违规标签/话题
- 画面/音频质量问题

---

## 二、发布规则

### 2.1 视频规格

| 参数 | 要求 |
|------|------|
| 时长 | 15秒 - 10分钟 |
| 分辨率 | 720P以上，推荐1080P |
| 格式 | MP4、MOV |
| 文件大小 | ≤500MB |
| 帧率 | 25-60fps |
| 编码 | H.264 |

### 2.2 图文规格

| 参数 | 要求 |
|------|------|
| 图片数量 | 1-20张 |
| 比例 | 9:16竖版最佳（3:4、1:1也可） |
| 分辨率 | ≥720×1280 |
| 格式 | JPG、PNG |
| 文件大小 | ≤20MB/张 |

### 2.3 封面规格

| 参数 | 要求 |
|------|------|
| 比例 | 9:16 (1080×1920) |
| 文字 | 大字、高对比、简洁 |
| 人物/产品 | 占画面60%以上 |
| 避免 | 留白过多、背景杂乱 |

---

## 三、标题规范

### 3.1 标题规则
- 长度：≤55字
- 必须为中文或中英文混合
- 避免违规词、敏感词
- 禁止标题党（夸大、虚假）

### 3.2 优质标题公式

**悬念型**：
```
不会还有人不知道XXX吧？
XXX竟然还能这样？
```

**痛点型**：
```
还在为XXX发愁？教你3招解决
XXX的5个误区，第3个最致命
```

**数字型**：
```
5个方法让你XXX
3分钟学会XXX
```

**情感型**：
```
XXX是我用过最好的
一定要XXX不然会后悔
```

---

## 四、正文/描述规范

### 4.1 描述规则
- 长度：0-500字
- 前3行最重要（决定是否继续看）
- 添加话题标签增加曝光
- 适当引导评论互动

### 4.2 描述结构示例

```
🎯 核心卖点/钩子（第一行）

📝 详细说明/使用场景

💬 互动引导（提问/投票）

#话题1 #话题2 #话题3
```

---

## 五、话题标签

### 5.1 标签规则
- 格式：#话题名（无空格）
- 数量：3-5个最佳
- 精准话题优先（搜索量高、相关度高）
- 泛话题辅助（曝光量大）
- 避免敏感话题

### 5.2 热门话题分类

**领域标签**：
- #美食 #美妆 #穿搭 #健身 #旅游
- #数码 #游戏 #音乐 #电影 #图书
- #职场 #情感 #亲子 #生活 #科技

**内容标签**：
- #vlog #日常 #教程 #测评 #种草
- #搞笑 #剧情 #挑战 #合集 #盘点

**时效标签**：
- #2024 #新年 #节假日 #热点事件

## 六、发布时段

### 6.1 最佳发布时间

| 时段 | 适用内容 |
|------|----------|
| 12:00-13:00 | 午休娱乐、轻松内容 |
| 18:00-20:00 | 下班放松、知识干货 |
| 20:00-22:00 | 黄金时段，流量最大 |
| 22:00-24:00 | 夜猫子活跃 |

### 6.2 注意事项
- 周中 vs 周末效果不同
- 避开重大事件/热点（会被分流）
- 新账号建议固定发布时间

---

## 七、算法推荐机制

### 7.1 流量池分级
1. **初始流量**：200-500播放
2. **二级流量**：3000-5000播放
3. **中级流量**：1-3万播放
4. **高级流量**：10万+播放
5. **爆款**：100万+播放

### 7.2 推荐指标权重
- **完播率**（最重要）：>50%有机会进入下一级
- **点赞率**：>3%较好
- **评论率**：>0.5%较好
- **转发率**：>0.3%较好
- **关注转化**：有粉丝关注更易推荐

### 7.3 CES评分
抖音使用CES评分：
```
CES = 播放时长 × 点赞 × 评论 × 转发 × 关注
```

## 八、常见问题

### 8.1 为什么播放量低？
- 标题/封面不够吸引
- 内容不够优质
- 违规被限流
- 账号权重低
- 发布时间不对

### 8.2 为什么被封禁？
- 违法违规内容
- 多次违规记录
- 恶意营销
- 刷量行为

### 8.3 如何提升权重？
- 优质内容持续输出
- 遵守平台规则
- 积极互动（评论、私信）
- 稳定发布频率
- 绑定更多个人信息
