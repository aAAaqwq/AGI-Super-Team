# 波段交易策略 (Swing Trading / 盯盘吃价差)

## 核心逻辑
在高波动市场中低买高卖，赚取中间价差。不需要预测最终结果，只需要捕捉短期波动。

## 选盘条件
1. **24h成交量 > $500K** — 流动性足够，能快速进出
2. **价格区间 20¢-80¢** — 波动空间大（太低太高波动小）
3. **到期 > 3天** — 有足够的波动周期
4. **近7天价格波动幅度 > 10¢** — 实际有波动

## 执行流程

### Step 1: 识别波动盘
```
GET gamma-api.polymarket.com/markets?active=true&closed=false&order=volume24hr&ascending=false&limit=50
```
筛选：
- volume24hr > 500000
- outcomePrices 在 0.20-0.80 之间
- endDate > 3天后

### Step 2: 分析支撑/阻力
- 查看近7天价格图（浏览器snapshot图表）
- 识别关键价位：反复触碰不突破 = 支撑/阻力
- 查看评论区舆论（多空比）

### Step 3: 挂单策略
- **买入：** 价格接近支撑位时市价买入
- **卖出：** 价格接近阻力位时市价卖出
- **止损：** 跌破支撑位5¢以下 → 立即割
- **止盈：** 到达阻力位前2¢ → 卖出

### Step 4: 仓位管理
- 单笔 ≤ 总资金10%
- 同时最多持有3个波段仓
- 盈利部分可加仓（金字塔加仓）

## 盯盘频率
- **活跃交易时段**（美东9AM-5PM = 北京22:00-06:00）：每30分钟检查
- **低波时段**：每2小时检查
- 价格触发条件时推送告警

## Cron任务模板
```
每30分钟扫描持仓市场价格变动：
1. 获取当前持仓列表
2. 对每个波段仓位：查当前价 vs 买入价
3. 触发条件：
   - 盈利 ≥ 目标 → 提醒卖出或自动卖出
   - 亏损 ≥ 止损线 → 立即卖出
   - 接近支撑位 → 提醒加仓机会
4. 推送到DailyNews群(newsbot_send.py) + Super Agents群
```

## 风险控制
- 不在到期前24h内开新波段仓（theta衰减太快）
- 重大新闻发布前平掉波段仓（避免跳空）
- 单笔最大亏损 = 投入金额的50%（超过立刻割）
- 日亏损上限 = 总资金5%（触发暂停交易）

## 适用市场类型
✅ 政治类（民调波动大）
✅ 地缘政治（消息面驱动波动）
✅ 加密相关（天然高波动）
❌ 体育类（结果太随机）
❌ 到期<3天（时间价值衰减太快）

## 实战模板
| 指标 | 买入信号 | 卖出信号 |
|------|---------|---------|
| 价格 | 触支撑位 ±2¢ | 触阻力位 -2¢ |
| K线历史高点 | — | 接近历史最高价且盈利 → 立即卖出 |
| 成交量 | 放量下跌后缩量 | 放量上涨接近阻力 |
| 舆论 | 恐慌/过度悲观 | 过度乐观/FOMO |
| 新闻 | 负面已Price in | 利好已Price in |
