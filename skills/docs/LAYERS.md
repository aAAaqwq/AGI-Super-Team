## 🔧 实现方式

### 目录结构
```
~/clawd/skills/
├── core/           # 核心层（软链接）
├── extended/       # 扩展层（软链接）
├── archive/        # 归档层（软链接）
└── [原始 skill 目录/软链接]
```

### 软链接示例
```bash
# 从 cc-skills 链接
ln -s ~/cc-skills/agent-builder ~/clawd/skills/agent-builder

# 分层链接
ln -s ../agent-builder ~/clawd/skills/extended/agent-builder
```
